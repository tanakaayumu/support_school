特進科コース

大学進学を目指すコース

コース特徴

国公立大・私立大ともに、近年の「思考力重視」の入試は専門の対策が必要です。このコースでは、志望校合格に向けたオーダーメイドカリキュラムで、専任講師によるマンツーマン指導を行います。

こんな人におすすめ

  • 志望校合格に向けてオーダーメイドのカリキュラムで指導を受けたい方
  • 最先端の技術も活用しながら効率よく学習したい方
  • 志望校の過去問対策を集中的にやりたい方

授業について

「思考力重視」の入試に対応

最新の受験情報・学校情報に精通した教育プランナーが、現状の学力や苦手科目・単元、性格まで考慮してオーダーメイドのカリキュラムを作成したうえで完全マンツーマン授業を行うため、志望校合格に向けた最短ルートを進むことができます。

「わかったつもり」を防ぐ「ダイアログ学習法」で弱点克服

授業は講師が一方的に説明するのではなく、生徒自身が教わった内容を自らの言葉で講師に説明し直す「ダイアログ学習法」を導入しています。「ダイアログ学習法」による双方向の対話を行うことで、生徒の理解があいまいな部分や、つまずきの原因を明らかにすることができます。

専任の講師による「完全マンツーマン指導」

常に同じ講師が授業を行う「専任制」を採用しています。担当生徒の性格や科目・単元ごとの得意/苦手を理解した上で授業を完全マンツーマン形式で行うからこそ、効率良く学習を進めることができます。

「トライ式学習AIタブレット」で効率良く得点力アップ

最先端のAIを搭載した「トライ式5教科学習AIタブレット」で、毎日の復習から定期テスト対策まで効率よく学習することができます。1科目わずか10分程度で自分の苦手と得意を分析できる「診断AI」を活用することで、苦手な科目・単元を特定できます。

学習プランについて

志望校の入試傾向に対応したカリキュラムを作成します。最新の受験情報に精通した教育プランナーが、過去の模試やテストの結果、授業ノートなどを分析しして志望校合格に最適なカリキュラムを作成します。定期的な学習面談を通して、学習進捗を確認やカリキュラムの修正を行います。

科目・時間

開講科目：英語 / 数I・A / 数Ⅱ・B / 数Ⅲ / 現代文 / 古典 / 化学 / 物理 / 生物 / 日本史 / 世界史 / 地理 / 政治経済 他

授業時間：生徒の目標や現状の学力に合わせて、週あたりの時間数や１回の授業時間を提案します。

トライ式高等学院の特徴・指導方針

トライ式高等学院の最大の特徴は、一人ひとりのペースや取り組み方・やる気に合わせた指導と寄り添いを徹底するということです。通常の教育現場では、先生：生徒＝1：多数で学習指導を行いますが、当校では完全個別指導による先生：生徒＝1：1の環境を実現しています。

また、教室で学ぶだけでなく、自分の学習スタイルや状況に合わせて、自宅でも効率よく学習できるよう、AIを活用したオンライン授業も用意。生徒一人ひとりのためのテストや学習プログラムの作成などが可能なほか、オンライン上で生徒と講師のコミュニケーションをとることも。非常時も含め、さまざまな状況下で学びが途切れない環境づくりを大切にしています。

不登校や引きこもりなどの理由で学校に通えなくなってしまった生徒でも、通信制高校へ進学することで居場所ができ、ひとりでも多くの生徒さんに自信と笑顔を取り戻してほしいという想いから、スタッフは生徒さんの話を常に真剣に聞く姿勢を持ち続けています。トライ式高等学院は「生徒の可能性を広げる場所」でありたいと考えています。

トライ式高等学院が選ばれる理由

トライ式高等学院が選ばれる第一の理由は、「完全個別指導」です。教師一人に対して生徒一人という環境で、生徒一人ひとりの理解度や学習進度、得意・不得意に合わせた指導を行っています。また、独自のインターネットアクセスによる24時間自主学習で、わからないポイントもすぐに確認することができています。

第二に、各分野の専門講師によるきめ細かな指導です。大学受験に精通した講師陣が、志望校合格へ導くカリキュラムを提供。高校卒業後の進路実現をしっかりサポートしています。

第三に、全国どこでも学べる機関であることです。引っ越しなどで転校が必要になっても、全国の校舎で学び続けることが可能です。教育のクオリティを保ちながら、生徒のライフスタイルに柔軟に対応しています。行事などを通じた交友関係の構築も大切にしており、不登校からの復学・受験合格に向けて包括的にサポート。心理カウンセラーの常駐など、メンタル面のケアも充実しています。

トライ式高等学院のスクールライフ
